Experiment about generating walls from OpenStreetMap indoor data.

Use the numbered buttons on the right to switch levels. Buttons are only shown for the indoor data visible in the current map view.

Data is pulled from Overpass API which gives world wide converage, kindly hosted by VK. Indoor maps are only available for a few buildings yet. See Simple Indoor Tagging in the OSM wiki to learn how to add more.

Source code for this map is available at github.com/AugustusKling/osm-floorplans. Note that this is an experiment only and not something to be reused.
